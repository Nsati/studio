# **App Name**: Uttarakhand Getaways

## Core Features:

- User Authentication: Secure user registration, login/logout with email and password using Firebase Authentication. Supports user and admin roles with protected routes.
- Hotel Browsing & Search: Browse hotels city-wise or search by location, check-in/check-out dates, number of guests and rooms.
- Hotel Details & Availability: Display hotel details with an image gallery, amenities, room types, and prices. Show real-time room availability.
- Real-time booking & Payment: Book rooms based on date availability. Lock the selected room for 5 minutes during payment to avoid overbooking. Make the stripe payment.
- Booking confirmation: Automatically send booking confirmation via email.
- Booking Management: Allow users to view, manage, and cancel their bookings.
- Admin Hotel Management: Admin dashboard to add, edit, and delete hotels. Includes features to add room types with price per night, capacity and total rooms, and upload hotel images. View and manage bookings.

## Style Guidelines:

- Primary color: Forest Green (#388E3C) to reflect the natural beauty of Uttarakhand.
- Background color: Light beige (#F5F5DC) to give a warm and inviting feel.
- Accent color: Earthy Orange (#D35400) for calls to action and highlights, reminiscent of sunsets in the mountains.
- Headline font: 'Playfair', serif, for a fashionable and high-end feel.
- Body font: 'PT Sans', sans-serif, to complement headline font and improve readability, for blocks of texts.
- Use icons that represent Uttarakhand tourism elements like mountains, rivers, and adventure activities.
- Design a clean, modern, and mobile-first responsive layout that is easy to navigate.
- Add subtle loading and transition animations to improve user experience.
